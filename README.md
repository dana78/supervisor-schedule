# Cronograma de Supervisores (React + JS)

Aplicación web para generar un cronograma de 3 supervisores con:
- Inputs: régimen W x R, inducción (1..5), días totales a perforar
- Tabla/grilla con colores
- Fila #P (supervisores perforando por día)
- Validaciones: nunca 3 perforando, mantener #P=2 desde el inicio de perforación, patrones inválidos

## Requisitos
- Node.js 18+

## Ejecutar local
```bash
npm install
npm run dev
